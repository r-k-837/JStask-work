JavaScript Interview Q&A

Array Methods

1. What is the difference between `push()` and `unshift()`?
`push()` adds an element at the end, while `unshift()` adds it at the beginning.
2. How does `splice()` work?
`splice(start, deleteCount, item1, item2, ...)` modifies an array by adding/removing elements.
3. What does `slice()` do?
Returns a shallow copy of a portion of an array without modifying the original.
4. How to check if an array includes a specific value?
Use `includes(value)`, `indexOf(value)`, or `some(callback)`.
5. Difference between `pop()` and `shift()`?
`pop()` removes the last element, `shift()` removes the first element.
6. What does `concat()` do?
Merges two or more arrays without modifying the original arrays.
7. How does `reverse()` work?
Reverses the order of elements in an array.
8. How to find the index of an element in an array?
Use `indexOf(value)`.
9. How to join array elements into a string?
Use `join(separator)`, default separator is a comma.
10. Difference between `every()` and `some()`?
`every()` checks if all elements satisfy a condition, `some()` checks if at least one does.

Object Methods

11. How to get all keys of an object?
Use `Object.keys(obj)`.
12. How to get all values of an object?
Use `Object.values(obj)`.
13. How to merge two objects?
Use `Object.assign(target, source)` or `{ ...obj1, ...obj2 }` (spread operator).
14. How to check if an object has a specific property?
Use `obj.hasOwnProperty('property')` or `'property' in obj`.
15. How to freeze an object?
Use `Object.freeze(obj)`, which prevents modifications.
16. How to seal an object?
Use `Object.seal(obj)`, which allows modifications but prevents adding/removing properties.
17. How to convert an object into an array?
Use `Object.entries(obj)`.
18. How to clone an object?
Use `{ ...obj }` (spread operator) or `Object.assign({}, obj)`.
19. How to get an object's prototype?
Use `Object.getPrototypeOf(obj)`.
20. How to check if two objects are equal?
Use `JSON.stringify(obj1) === JSON.stringify(obj2)`.
21. How to create an object without a prototype?
Use `Object.create(null)`.
22. What does `Object.defineProperty()` do?
Defines a new property with specific descriptors like `writable`, `configurable`.
23. What is `Object.getOwnPropertyDescriptors()`?
Returns all property descriptors of an object.
24. How to copy an object deeply?
Use `structuredClone(obj)` or `JSON.parse(JSON.stringify(obj))`.
25. What does `Object.is()` do?
Checks if two values are the same, similar to `===` but handles NaN and -0 correctly.
